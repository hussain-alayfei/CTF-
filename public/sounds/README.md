# First-blood sound

Drop your custom first-blood sound effect in this folder as either:

- `first-blood.mp3`, or
- `first-blood.wav`

The app automatically tries `first-blood.mp3` first, then `first-blood.wav`.
If neither file is present (or it fails to load), the app falls back to its
built-in synthesized siren sound automatically — nothing else needs to change.

Recommended: a short (2-5 second) dramatic sting/alarm clip. Keep the file
size reasonable (under ~1 MB) since it is downloaded by every player's
browser the moment a first blood happens.
